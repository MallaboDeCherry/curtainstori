# Умные шторы (макет) — Smart Curtain Prototype

Прошивка для управления умной шторой на базе **ESP32**.  
Проект реализует полноценное управление шаговым двигателем через **MQTT** и **REST API** в составе системы "Умная лаборатория Комфорт".

Первоначальная настройка Wi-Fi осуществляется через **BLE**. После подключения к сети устройство становится доступно через **MQTT-брокер** и **REST API** сервера.

---

## 🚀 Основные возможности

- Управление шаговым двигателем (открытие / закрытие / позиционирование 0-100%)
- Неблокирующая реализация движения (не тормозит основной цикл)
- **MQTT** — управление и отправка статуса в реальном времени
- **REST API** — совместимость с мобильным приложением и веб-интерфейсом
- **BLE** — первичная настройка Wi-Fi
- Состояния шторы: `OPENED`, `CLOSED`, `MOVING`, `STOPPED`
- Принудительная остановка движения
- Автоматическое переподключение к Wi-Fi и MQTT
- Сохранение позиции в энергонезависимую память (Preferences)
- Публикация статуса в реальном времени (каждые 500 мс при движении)
- Поддержка уникального ID для каждого устройства

## 📡 Протоколы взаимодействия

| Протокол | Назначение |
|----------|------------|
| **MQTT** | Передача команд управления и статуса между ESP32 и сервером |
| **REST API** | Управление через мобильное приложение и веб-интерфейс |
| **BLE** | Первичная настройка Wi-Fi (SSID, пароль, Device ID) |

### MQTT Топики

| Топик | Направление | Описание |
|-------|-------------|----------|
| `blinds/{device_id}/set` | Сервер → ESP32 | Команда управления (позиция 0-100%) |
| `blinds/{device_id}/position` | ESP32 → Сервер | Текущая позиция шторы |
| `devices/announce` | ESP32 → Сервер | Приветственное сообщение при подключении |

### Формат команды через BLE

SSID:ПАРОЛЬ:DEVICE_ID

Пример:
A177-COMP:password123:blind_642fc62f2b14

## 🔌 Аппаратное обеспечение

| Компонент | Модель |
|-----------|--------|
| Микроконтроллер | ESP32 (DevKit V1 / NodeMCU) |
| Драйвер шагового двигателя | TB6600 (или L298N) |
| Шаговый двигатель | Nema17 |

### Подключение (TB6600)

| ESP32 | TB6600 |
|-------|--------|
| GPIO 18 | DIR- (направление) |
| GPIO 17 | PUL- (шаги) |
| GPIO 19 | ENA- (включение) |
| GND | GND |
| 5V | VCC |

### Подключение (L298N)

| ESP32 | L298N |
|-------|-------|
| GPIO 18 | IN1 |
| GPIO 17 | IN2 |
| GPIO 19 | ENA (PWM) |

## ⚙️ Установка и настройка

### 1. Зависимости (PlatformIO)

```ini
[env:esp32dev]
platform = espressif32
board = esp32dev
framework = arduino
monitor_speed = 115200
lib_deps = 
    knolleary/PubSubClient @ ^2.8
    bblanchon/ArduinoJson @ ^6.21.3


2. Настройка MQTT
В MqttManager.cpp укажите IP сервера:

const char* mqtt_server = "192.168.2.112";
const int mqtt_port = 1883;

-----

3. Загрузка прошивки

pio run --target upload

---------------------

4. Первичная настройка через BLE

1. Включите ESP32 — LED начнёт медленно мигать (BLE режим)
2. Откройте мобильное приложение "Умные шторы" → "Добавить устройство"
3. Выберите ESP32 из списка BLE-устройств
4. Введите SSID и пароль вашей Wi-Fi сети
5. ESP32 подключится к Wi-Fi и MQTT

-----

📱 Управление/

Через MQTT
# Установить позицию 50%
mosquitto_pub -t "blinds/blind_642fc62f2b14/set" -m "50"

# Открыть (100%)
mosquitto_pub -t "blinds/blind_642fc62f2b14/set" -m "100"

# Закрыть (0%)
mosquitto_pub -t "blinds/blind_642fc62f2b14/set" -m "0"

-------

Через REST API
# Установить позицию 75%
curl -X POST http://192.168.2.112/api/blind/control \
  -H "Authorization: Bearer <token>" \
  -H "Content-Type: application/json" \
  -d '{"deviceId":"blind_642fc62f2b14","position":75}'
