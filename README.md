## Описание

Прошивка для управления умной шторой на базе **ESP32**.  
Проект реализует полноценное управление шаговым двигателем через Wi-Fi с помощью HTTP API.  

Первоначальная настройка Wi-Fi осуществляется через **BLE**. После подключения к сети устройство становится доступно по адресу `http://smart-curtain.local` (mDNS).

## Основные возможности

- Управление шаговым двигателем (открытие / закрытие шторы)
- Неблокирующая реализация движения (не тормозит основной цикл)
- HTTP API, совместимое с мобильным приложением / системами умного дома
- Состояния шторы: `OPENED`, `CLOSED`, `MOVING`
- Принудительная остановка движения
- Подготовка к будущим расширениям: концевики, позиционирование, регулировка скорости

## Недавние изменения

**Добавлена реализация управления мотором шторы (CurtainControl)**

- Реализован интерфейс `CurtainControl.h`:
  - `initCurtain()` — инициализация пинов драйвера
  - `openCurtain()` — запуск открытия (неблокирующий)
  - `closeCurtain()` — запуск закрытия (неблокирующий)
  - `getCurtainStatus()` — текущее состояние
  - `stopCurtain()` — немедленная остановка

- Управление шаговым двигателем через пины:
  - `DIR_PIN` — направление
  - `STEP_PIN` — импульсы
  - `ENABLE_PIN` — включение/отключение драйвера

- Неблокирующая генерация STEP-сигнала с использованием `micros()`
- Таймер на `millis()` для ограничения максимального времени движения
- Функция `updateCurtain()` вызывается в `loop()` для продолжения движения
- Автоматическое обновление состояния по завершении движения
## Аппаратное обеспечение

- Микроконтроллер: **ESP32**
- Драйвер шагового двигателя tb6600
- Мотор Nema17
